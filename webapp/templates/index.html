Configuration Reseau


            

                

                    

                        Proxy Transparent
                        
                    

                    

                        Redirige le trafic HTTP/HTTPS via iptables vers mitmproxy (port 8080).
                    

                


                

                    

                        Mitmproxy Web UI
                    

                    

                        Interface web de mitmproxy pour visualiser le trafic intercepte.
                    

                    Ouvrir Mitmproxy
                


                

                    

                        Certificat Mitmproxy
                    

                    

                        Installer le certificat CA sur l'appareil Android (root requis) pour intercepter le HTTPS.
                    

                    
adb root
adb remount
adb push certs/*.0 /system/etc/security/cacerts/
adb shell chmod 644 /system/etc/security/cacerts/*.0
adb reboot

                


                

                    

                        IP Forwarding
                    

                    

                        Active automatiquement avec le proxy transparent.
                    

                    
sysctl net.ipv4.ip_forward=1
